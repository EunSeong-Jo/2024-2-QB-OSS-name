## 📋 프로젝트 소개
### 프로젝트 이름: **WhyGroup project**
**WhyGroup project** 는 오픈소스소프트웨어에 대해 알아보고 Github의 여러 기능들을 사용해보며 초보자들이 오픈소스소프트웨어와 Github에 더욱 친근하게 다가갈 수 있도록 만드는 프로젝트입니다.

#### 주요 내용
- OSS 개요
- OSS 역사
- OSS 소개
- OSS 저작권
- 깃허브 기능 소개
